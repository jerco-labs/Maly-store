# MalyStore - Parte 4.4

Esta parte incluye un dashboard funcional para el proyecto **MalyStore**.

## Archivos incluidos
- `pages/dashboard.html`
- `css/dashboard.css`
- `js/dashboard.js`
- `README_PARTE_4_4.txt`

## Qué muestra
- Resumen general con tarjetas KPI.
- Gráfico de ventas semanal.
- Lista de productos top.
- Tabla de pedidos recientes con buscador.
- Alertas rápidas del estado del negocio.

## Cómo usarlo
1. Extrae el ZIP dentro de la carpeta principal del proyecto `MalyStore`.
2. Abre `pages/dashboard.html` en el navegador.
3. Verifica que las rutas relativas a `../css/dashboard.css` y `../js/dashboard.js` se mantengan intactas.

## Observación
El dashboard está hecho sin librerías externas para que funcione de forma simple y directa en cualquier entorno web básico.
