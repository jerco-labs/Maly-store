- pages/admin.html
- css/admin.css
- css/toast.css

Qué hace:

Instalación:
1. Extrae este ZIP dentro de tu carpeta MalyStore.
2. Reemplaza los archivos existentes si Windows lo pide.
3. Abre pages/admin.html con Live Server.
