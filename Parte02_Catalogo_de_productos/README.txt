# Maly'store - Parte 2

Esta entrega incluye:
- `index.html`
- `pages/categoria.html`
- `pages/carrito.html`
- `pages/contacto.html`
- `pages/nosotros.html`
- `css/style.css`
- `js/app.js`
- `data/productos.json`
- `manifest.json`
- `service-worker.js`
- SVGs de apoyo en `img/`

## Cómo usar
1. Extrae el ZIP dentro de tu carpeta `MalyStore`.
2. Abre `index.html` en VS Code.
3. Ejecuta con Live Server si lo tienes.
4. Edita solo `data/productos.json` para cambiar productos.

## Nota
Los QR y el WhatsApp están preparados como ejemplos y se pueden reemplazar con tus datos reales.
